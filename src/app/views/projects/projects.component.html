Фриланс Агенство
Веб-приложение, разработанное на Angular, предназначенное для поиска исполнителей на фриланс-услуги. Пользователь может оставить заявку с описанием задачи, после чего с ним свяжется менеджер для уточнения деталей. Проект включает блоговую часть с новостями и статьями, где доступны функции комментирования, лайков и взаимодействия с контентом.

Дашборд Финансов
Веб-приложение дашборд для учёта личных доходов и расходов. Разработано на TypeScript. Приложение позволяет вести записи о доходах и расходах, просматривать финансовую статистику в виде диаграмм, категоризировать операции, добавлять, редактировать и удалять операции, фильтровать их по дате и типу, а также отслеживать общий баланс и суммы.

Квиз по JavaScript
SPA-приложение, разработанное на TypeScript. Проект реализует функциональность онлайн-тестирования с возможностью регистрации и авторизации пользователей. Приложение взаимодействует с локальным бэкендом, откуда получает список тестов и вопросов. После прохождения теста пользователю отображается результат.
Все основные действия — регистрация, вход, выбор теста, прохождение и получение результата — происходят без перезагрузки страницы, благодаря архитектуре SPA.
